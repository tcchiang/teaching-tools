教學概念提醒

1. 所有檔案在電腦裡，本質上都是一串 bytes。

2. 所謂 text file，是這些 bytes 可以用某種文字編碼合理解讀。

3. 所謂 binary file，通常不是設計給文字編輯器直接閱讀，因此文字區可能顯示亂碼。

4. Hex / Byte View 比較接近檔案的原始儲存樣貌。
